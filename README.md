# RailWatch AI — Smart Railway Crowd Monitoring System

A hackathon project built to address real-time overcrowding issues in Indian Railways using AI-powered complaint analysis.

## Problem Statement

Reserved coaches in Indian trains frequently face unauthorized occupancy and severe overcrowding, especially during peak hours. Railway authorities lack a fast, structured system to identify and prioritize such complaints.

## Solution

RailWatch AI allows passengers to submit overcrowding complaints which are instantly analyzed by Google Gemini AI. The system returns a structured report — problem category, crowd severity, risk level, and a recommended action — helping railway authorities respond faster and smarter.

## Features

- Passenger complaint submission with coach number and train route
- AI-powered analysis using Google Gemini API
- Crowd severity classification: Low, Medium, High
- Risk level assessment and recommended railway action
- Live coach occupancy monitoring dashboard
- Fully responsive design

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Google Gemini 1.5 Flash API

## Project Structure

## How to Run

1. Clone this repository
2. Open script.js and paste your Gemini API key on line 10
3. Open index.html in any browser
4. No installations or build tools required
